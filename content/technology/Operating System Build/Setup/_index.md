## Using Windows Subsystem for Linux (Debian)
- Install list
	- gcc
	- build-essential
	- bison 
	- flex 
	- libgmp3-dev 
	- libmpc-dev 
	- libmpfr-dev 
	- texinfo
- Cross-Compiler
	- `git clone https://github.com/lordmilko/i686-elf-tools`
	- `cd i686-elf-tools`
	- `./i686-elf-tools.sh linux`
